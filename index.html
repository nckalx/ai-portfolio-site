    Phase 3 Output

    Executive Summary

    Movement is calculated by comparing each mapped baseline start/finish date to the mapped current start/finish date. A task can have a multi-day duration and still be Unchanged if its baseline and current dates are the same.

    Likely parent/summary rows are counted separately and will be excluded from future critical path analysis by default. When they appear in changed-row or dependency output, they remain visible so schedule reviewers can inspect the source rows.

    Phase 3 Output

    Task Movement Detail

    Showing changed schedule rows only. Unchanged rows and warning-only rows are included in the downloadable Excel report.

    Rows with invalid mapped dates stay in the table, but their movement values are not calculated.

    Dependency Check

    Dependency / Predecessor Validation

    Predecessor values are parsed for dependency quality checks only. Critical path calculation has not been added yet.

    Excel Report

    Download Summary Workbook

    The Excel report includes changed rows, all analyzed rows, warning rows, dependency validation, and the column mapping used.

    Future Phase

    Not included in Phase 3

    Full critical path calculation and holiday calendars are intentionally reserved for later phases.
